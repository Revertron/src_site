    А чем качать?

    • Скачивать c YouTube лучше всего с Ummy Downloader, для пердоликов youtube-dl.

    • Для скачивания рандомных сайтов подойдёт Teleport Pro (Есть ещё версия ULTRA, которая добавляет возможность запихнуть в .txt файл до 4094 ссылки и скачать их), прога крайне проста, но нужно аккуратно вбивать, чтобы не скачать пол интернета. Из аналогов есть HTTrack Website Copier, WebCopy, Darcy Ripper, MetaProducts Offline Explorer. Вот Teleport Pro крякнутая и ру$$сифицированная, пароль к архиву kilo. Для линукса можно юзать wget -mkEpnp http://example.org. Для дампа всяких “Index of /” есть wget -m -np -c -R “index.html*” “http://example.org".

    • Для скачивания шебемочек с различных сайтиков (Instagram, tumblr, YouTube, SoundCloud mp3, Facebook, VK, Xvideos, Putlocker, Vimeo …) подойдёт Tube Offline

    • Если ты только-что очнулся и уже ПОЧАЛОСЬ, то скачивай DC++.
